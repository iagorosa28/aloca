# Log de decisões

Este arquivo registra decisões relevantes do projeto Aloca.

## 2026-04-29 - Aloca como laboratório de desenvolvimento com IA

### Decisão

O Aloca será desenvolvido como uma aplicação de organização de carteira de investimentos e também como um laboratório pessoal de desenvolvimento assistido por IA.

### Contexto

O projeto está em fase inicial e já possui uma estrutura base com `backend/`, `frontend/` e `e2e/`. Além do objetivo funcional do produto, foi decidido que o processo de desenvolvimento também será parte explícita do aprendizado.

### Direção definida

- ChatGPT será usado para planejamento, discussão, revisão, arquitetura, prompts e documentação.
- Codex será usado como agente principal de implementação, investigação do código, alterações em arquivos e documentação.
- Iago não pretende escrever código manualmente neste projeto.
- Iago continuará responsável por decisões, validação, testes, revisão e aprovação das mudanças.
- Mudanças geradas por IA não devem ser aceitas automaticamente.
- Decisões importantes devem ser documentadas conforme o projeto evoluir.

### Consequências

Essa decisão torna o processo de desenvolvimento parte do próprio projeto. Além de construir o produto, o Aloca também servirá para experimentar, avaliar e documentar formas práticas de trabalhar com IA no ciclo de desenvolvimento de software.

### Observação

O Aloca continua sendo uma ferramenta de organização e simulação baseada em dados informados pelo usuário. A aplicação não realiza recomendação financeira.
