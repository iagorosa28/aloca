# Aloca

Aplicação web para organização de carteiras de investimento e simulação de aportes com base em alocação percentual.

## Objetivo

O Aloca tem dois objetivos principais:

1. Criar uma ferramenta simples para ajudar usuários a visualizar a distribuição da própria carteira e simular aportes com base em percentuais desejados.
2. Servir como laboratório pessoal de desenvolvimento assistido por IA, utilizando ChatGPT para planejamento e revisão técnica, e Codex para implementação.

## Premissa de desenvolvimento

Neste projeto, o código é gerado por agentes de IA, enquanto as decisões de produto, validação, testes, revisão e aprovação são feitas manualmente.
