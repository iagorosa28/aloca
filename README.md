# 🚀 Aloca

O **Aloca** é um projeto pessoal criado com foco em **aprendizado**, **evolução técnica** e **construção de portfólio**.

A proposta do projeto é desenvolver uma aplicação web para **organização de carteiras de investimento** e **planejamento de aportes** com base em **alocação percentual**, buscando unir utilidade prática com uma construção técnica bem estruturada.

## 📌 Sobre o projeto

Este repositório está em fase inicial de desenvolvimento.

Neste momento, o projeto ainda está sendo estruturado, com definição da base da aplicação, organização do repositório e preparação do ambiente de desenvolvimento.

## 🎯 Objetivos

- desenvolver um projeto full stack com aplicação real
- praticar arquitetura, organização e boas práticas de desenvolvimento
- aplicar testes e evoluir a qualidade do código ao longo do tempo
- construir um projeto relevante para portfólio

## 🛠️ Status

Projeto em desenvolvimento inicial.
